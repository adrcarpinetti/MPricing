# MPricing — Plataforma de Preciação Montreal

Sistema web de composição de preços para propostas comerciais.

## Acesso padrão
- **Usuário:** `admin`
- **Senha:** `M0ntreal@01`

> ⚠️ Altere a senha no primeiro acesso via menu do usuário.

## Perfis
| Perfil | Acesso |
|--------|--------|
| Administrador | Total |
| Pricing/Controladoria | Propostas, parâmetros, relatórios |
| Comercial | Criação de propostas |
| Gestor | Relatórios |
| Leitor | Somente consulta |
